Liquid Cooler คืออะไร เย็นจริงหรือแค่สวยงาม

หลายๆคนอาจจะรู้อยู่แล้วว่าคอมพิวเตอร์นั้นเมื่อมีการใช้งานย่อมต้องเกิดความร้อนขึ้นมาในการใช้งาน ซึ่งเมื่อเกิดความร้อนแล้ววิธีระบายความร้อนก็คือ การติดตั้ง Air cooler นั่นเอง แต่บางครั้ง Air cooler แบบพัดลมที่เราเห็นกันมันอาจจะช่วยระบายความร้อนได้ไม่พอ จึงมีการผลิต Liquid Cooler หรือระบบระบายความร้อนด้วยน้ำนั่นเอง วันนี้ ComputeAndMore จะมาพูดถึงระบบระบายความร้อนแบบน้ำกัน

ระบบระบายความร้อน

ระบบระบายความร้อนของ CPU ( CPU Cooling System ) ในเครื่อง PC นับเป็นอีกหนึ่งอุปกรณ์ที่มีความสำคัญมากต่อคอมพิวเตอร์เนื่องมาจากการทำงานของ CPU นั้นย่อมทำให้เกิดความร้อนขึ้น เมื่อมีความร้อนเกิดขึ้นแต่ไม่มีการระบายความร้อนก็จะทำให้ประสิทธิภาพในการทำงานลดลงและจะตามมาด้วยอาการที่คุ้นเคยเช่น คอมรวม ค้าง หรือ อาการผิดปกติของคอมพิวเตอร์อื่นๆนั่นเอง ดังนั้นจึงต้องมีตัวช่วยในการระบายความร้อนของ CPU นั่นเอง

ระบบระบายความร้อนมีอะไรบ้าง

ระบบความร้อนของ CPU มีด้วยทั้งหมด 3 แบบ คือ ระบบระบายความร้อนด้วยอากาศ ( Air Cooling ) , ระบบระบายความร้อนด้วยน้ำ ( Liquid Cooler ) และ ระบบระบายความร้อนด้วยไนโตรเจนเหลว ( Liquid Nitrogen Cooling System ) สำหรับ Air cooler และ Water Cooling นั้นสามารถเห็นได้ทั่วไปมีตั้งแต่ราคาถูกไปจนแพง แต่สำหรับตัวไนโตรเจนเหลวนั้นไม่ค่อยเป็นที่ใช้งานในหมู่ User มากนัก ส่วนมากจะใช้กับนัก Overclock ที่ต้องการให้ระบบ Clock ได้ประสิทธิภาพสูงๆแทนนั่นเอง แต่ในบทความนี้ เราจะมาพูดถึงเรื่อง “ระบบระบายความร้อนด้วยน้ำ” กัน

ระบบระบายความร้อนด้วยน้ำ (Liquid Cooler)

ระบบระบายความร้อนด้วยน้ำ ( Water Cooling System ) เป็นระบบระบายความร้อน CPU ที่ถูกพัฒนาขึ้นมาทดแทน ระบบระบายความร้อนด้วยอากาศ ( Air Cooling System ) โดยระบบระบายความร้อนด้วยน้ำจะใช้ ของเหลว หรือ สารหล่อเย็น ในการช่วยระบายความร้อนออกจาก CPU โดยตรง และ นำความร้อนระบายออกทางฟินอะลูมิเนียม หรือ หม้อน้ำ ( Radiator ) แล้วจะมีพัดลมช่วยเป่าไล่อากาศร้อนออกไปนั่นเอง โดยบางคนอาจจะได้ยินว่าหม้อน้ำ 1 ตอน 2 ตอน หรือ 3 ตอน มันคือ ขนาดของฟินพร้อมพัดลมที่เป่าระบายอุณหภูมินั่นเอง โดยระบบระบายความร้อนด้วยน้ำนั้นจะมีอยู่ 2 ประเภทด้วยกัน คือ ระบบระบายความร้อนด้วยน้ำปิด และ ระบบระบายความร้อนด้วยน้ำเปิด

ระบบระบายความร้อนด้วยน้ำปิด ( Closed Loop Water Cooling System )

ระบบระบายความร้อนด้วยน้ำแบบ “สำเร็จรูปพร้อมใช้งานได้ทันที” ซึ่งเราสามารถหาซื้อได้ตามร้านค้าทั่วไปมาประกอบและใช้งานเข้ากับเมนบอร์ดและอุปกรณ์ของเราได้โดยทันทีไม่ยุ่งยาก ภายในของระบบชุดน้ำปิดนั้นจะมีของเหลวที่เรียกว่าน้ำคูลแลนท์ หรือ น้ำยาหล่อเย็น หลายคนๆอาจจะเข้าใจว่ามันเป็นสารชนิดเดียวกันกับน้ำหล่อเย็นที่เราใช้เติมหม้อน้ำหน้ารถของเรา จริงๆแล้วไม่ใช่เนื่องจากน้ำหล่อเย็นในชุดระบายความร้อนนั้นมีคุณภาพที่ดีกว่ามากและไม่เกินตะกรัน (ตะกรันคือธาตุที่มารวมตัวกันเป็นก้อนแข็งยึดเกาะตามผิวของอุปกรณ์ซึ่งมีความร้อน โดยปกติธาตุเหล่านี้จะละลายอยู่ในน้ำ เมื่อผ่านผิวของอุปกรณ์ที่มีความร้อน จะเกิดการเปลี่ยนสถานะก่อตัวยึดกันแน่นตามผิวของอุปกรณ์เกิดเป็นตะกรันแข็งขึ้นมานั่นเอง) ซึ่งชิ้นส่วนของระบบชุดน้ำปิดจะมีอยู่ 3 ส่วนใหญ่ๆ คือ หม้อน้ำ ( Radiator ) , สายยาง ( Tubing ) และ ปั๊มน้ำ ( Pump ) โดยส่วนมากตัวปั๊มน้ำจะถูกรวมอยู่กับหน้าสัมผัสทองแดงที่ใช้ระบายความร้อนของ CPU นั่นเอง ระบบการทำงานคือน้ำไหลผ่าน Surface ที่สัมผัสกับ CPU ถูกส่งไปที่หม้อน้ำผ่านสายยางเพื่อให้ระบายความร้อนผ่านฟินแล้วนำสารหล่อเย็นที่ถูกระบายความร้อนแล้วกลับมาหมุนเวียนในระบบอีกครั้งนั่นเอง โดยชุดน้ำปิดนั้นมีหลากหลายขนาด ราคา และ ยี่ห้อ โดยแบรนด์เด่นๆในชุดน้ำปิดนั้นจะมีอาทิเช่น NZXT, Thermaltake, Cooler Master, Corsair, Silver Stone เป็นต้น โดยในปัจจุบันชุดน้ำปิดจะมาพร้อมลูกเล่นไฟ ARGB และ ระบบปรับแสงผ่านซอฟแวร์เพิ่มความสวยงามให้กับ PC กันแล้ว

ระบบระบายความร้อนด้วยชุดน้ำเปิด ( Open Loop Water Cooling System )

ระบบระบายความร้อนด้วยน้ำแบบเปิดหรือที่เรียกกันสั้นๆว่า ชุดน้ำเปิด หมายถึง ระบบระบายความร้อนด้วยน้ำที่ผู้ใช้สามารถปรับแต่งรูปแบบได้ดั่งใจอิสระตรงตามต้องการ อีกทั้งยังเป็นที่นิยมในปัจจุบันเป็นอย่างมาก เพราะนอกจากปรับแต่งได้อิสระแล้ว ยังสวยงาม อีกทั้งยังสามารถทำระบบให้ระบายความร้อนให้ครบวงจรได้ไม่ว่าจะเป็น CPU , VGA , RAM พร้อมกันได้อีกด้วยแต่กว่าจะได้ระบบระบายความร้อนแบบน้ำมานั้นจะมีราคาและงบประมาณในการทำค่อนข้างสูง อีกทั้งยังต้องหาผู้ที่เชี่ยวชาญในการวางระบบน้ำ มาติดตั้ง ดัดงอท่อ และ ตัดแต่งให้ระบบน้ำนั้นใช้งานได้อีกด้วยเพราะถ้าหากมีส่วนใดส่วนหนึ่งของระบบเกิดการรั่วซึมหรือปิดไม่สนิทขึ้นมาอาจจะทำให้เกิดความเสียหายแก่คอมพิวเตอร์ของเราได้ เช่น ลัดวงจรไฟฟ้า ส่วนประกอบทั่วไปของระบบน้ำเปิดจะมีหลากหลายชิ้นส่วนไม่ว่าจะเป็น หม้อน้ำ ปั๊ม ถังพักน้ำ ฟิตติ้ง บล็อกน้ำ สายยาง ท่ออะคริลิค หรือท่อสแตนเลส อีกด้วย โดยแบรนด์ที่เป็นแบรนด์ดังสำหรับชุดน้ำเปิดนั้นจะประกอบไปด้วย Corsair Thermaltake เป็นต้น

ชุดน้ำเปิด vs ชุดน้ำปิด แบบไหนดีกว่ากัน

หลังจากที่เราพอทราบแล้วว่าชุดน้ำเปิดและชุดน้ำปิดนั้นคืออะไร แตกต่างกันอย่างไร เราจะมาพูดถึงสิ่งที่คนถามถึงว่า สรุปแล้วชุดน้ำปิดหรือชุดน้ำเปิดอะไรมันดีกว่ากัน ถ้าให้พูดโดยอิงความคิดผมแล้วทั้งแบบมีข้อดีและข้อด้อยที่แตกต่างกัน โดยชุดน้ำปิดนั้นจะมีจุดเด่นในเรื่องของความสะดวกสบาย จ่ายทีเดียวจบ ระบบออกแบบมาให้พร้อมแล้วว่าต้องใช้อะไรบ้าง ใส่แล้วไม่จำเป็นต้องสนใจว่าน้ำระเหยหมดหรือยัง ต้องระวังเรื่องฝุ่นไหมอีกทั้งยังสามารถปรับระบบไฟ RGB สีสัน เราไปถึงดัดแปลงการวางหม้อได้ตามใจชอบ การระบายความร้อนนั้นก็จะอยู่ในจุดที่ดี-ดีมากแล้วแต่ราคาและสเป็คของชุดน้ำปิดนั้นๆ ต่างกับ ชุดน้ำเปิดที่ต้องใช้ความละเอียดมากกว่าเนื่องจากมีหลายๆเรื่องที่จะต้องระวังไม่ว่าจะเป็นท่อรั่ว ท่อแตก แรงดัน ระบบน้ำ น้ำในหม้อพัก แต่ผลลัพธ์ที่ได้ในการระบายความร้อนนั้นถือว่าดีกว่าชุดน้ำปิดก็เพราะ ชุดน้ำแบบเปิดนั้นสามารถเลือกได้ว่าอยากจะระบายส่วนใดบ้าง เย็นขนาดไหนเพิ่มได้ตามต้องการนั่นเอง ถือว่าดีกันทั้งคู่แต่ดีกันคนละแบบนั่นเอง

สรุปสุดท้ายแล้ว

ชุดน้ำนั้นก็เป็นอีกหนึ่งทางเลือกกับเราสำหรับใครที่ต้องใช้งานคอมพิวเตอร์ที่มีสเป็คระดับนึง ใช้งานค่อนข้างหนัก ไม่ต้องการให้คอมพิวเตอร์ร้อนจนเสื่อมสภาพ การติดตั้งชุดน้ำจึงสำคัญมากๆนั่นเองดังนั้นการเลือกซื้อชุดน้ำระบายอากาศควรจะเป็นอีกหนึ่งส่วนที่เราต้องให้ความสำคัญเช่นเดียวกับการเลือกสเป็คของคอมพิวเตอร์ของเรานั่นเองสำหรับใครที่สนใจสามารถสอบถามข้อมูลเพิ่มเติมรวมไปถึงออกแบบชุดน้ำเปิดได้อย่างเหนือระดับที่ ComputeAndMore เพราะที่นี่คืออีกระดับของคุณภาพ
